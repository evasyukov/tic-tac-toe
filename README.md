# Крестики нолики на React!

## Установка и запуск проекта
1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/evasyukov/tic-tac-toe
    ```

2. Перейдите в директорию проекта:
    ```bash
    cd tic-tac-toe
    ```

3. Установите зависимости:
    ```bash
    npm install
    ```

4. Запустите проект:
    ```bash
    npm run dev
    ```
